# RangeSlider component

Basic range component with 4 status:

 - Normal
 - Hover
 - Active
 - Disabled

The component has 4 properties for multiple use cases.
** If not provided, the default value for each property is automatically setted to 0 or False. **

## Properties

```typescript
type RangeSliderProps = {
    initialValue?: number;
    disabled?: boolean;
    min?: number;
    max?: number;
    step: number;
    onchange?: Function;
 };
}
```
