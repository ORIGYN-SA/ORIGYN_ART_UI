# Button component

Basic button component, for 3 button types:

 - primary (Default)
 - secondary (Bordered)
 - ternary (Text like)

You can look at visual appearance of the buttons in
[storybook style guides](https://www.figma.com/file/22ewljg1RGLMm50GA0J1KJ/Origyn-Open-Marketplace?node-id=696%3A2945)

## Properties

```typescript
type ButtonProps = {
  size?: "sm" | "md" | "lg";
  btnType?: "primary" | "secondary" | "ternary";
  disabled?: boolean;
  fullWidth?: boolean;
}
```

** IMPORTANT: Size prop is still in development **
