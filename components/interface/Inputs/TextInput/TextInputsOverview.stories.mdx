# TextInput component

Basic input for text. Can use ```value``` and ```onChange``` to control the input or be uncontrolled.

Available props:

```typescript
type TextInputProps = {
  label: string;
  name: string;
  error?: string;
  placeholder?: string;
  value?: string;
  onChange?: ChangeEventHandler;
}
```

```name``` is required as its used to provide id for input and connect it with label.
